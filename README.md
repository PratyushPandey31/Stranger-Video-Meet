# Stranger Video Meet 📹🌍

Welcome to **Stranger Video Meet** — a fully functional, real-time video conferencing application that brings people together. Whether you want to host private video meetings with your loved ones or seamlessly connect with strangers, this platform provides high-quality video and audio communication in a sleek, user-friendly interface.

## ✨ Key Features

- **Real-Time Video & Audio:** Crystal clear communication powered by WebRTC.
- **Guest Mode:** Join meetings instantly without the hassle of creating an account.
- **Secure Rooms:** Unique meeting links ensure your private conversations stay private.
- **Live Text Chat:** Send real-time messages within the meeting room.
- **Modern UI/UX:** A responsive, elegant design built with React and Material UI.

## 🛠️ Technology Stack

**Frontend:**
- React.js
- Material UI (MUI)
- Socket.io-client
- React Router DOM

**Backend:**
- Node.js
- Express.js
- Socket.io (for real-time signaling and chat)
- MongoDB (Mongoose for database operations)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PratyushPandey31/Stranger-Video-Meet.git
   cd Stranger-Video-Meet
   ```

2. **Install Backend Dependencies:**
   ```bash
   cd backend
   npm install
   ```

3. **Install Frontend Dependencies:**
   ```bash
   cd ../frontend
   npm install
   ```

### Running the Application

1. **Start the Backend Server:**
   Open a terminal in the `backend` folder and run:
   ```bash
   npm run dev
   ```
   *(The server will start on port 8000)*

2. **Start the Frontend Development Server:**
   Open a new terminal in the `frontend` folder and run:
   ```bash
   npm start
   ```
   *(The application will start on port 3000)*

3. **Open your browser:**
   Navigate to `http://localhost:3000` to start connecting!
